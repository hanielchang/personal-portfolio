About Me

Welcome to my portfolio. I am part of the UCSB alumni class of 2020. I studied Physics with an emphasis in electronics, including some software. While my interest in the theoretical side is strong, I found computer to be quite intriguing. Coding first appeared strange to me, but as I realized the power of it, I started to appreciate practicing it more and more. This inspired my motivation to pursue coding even after graduating UCSB. This led me to completing a full 6 months of coding bootcamp! I always look to improve my skillset and knowledge, as well as make new connections.


Apart from working and doing coding, I also like to play video games and dance. Halo is one of my favorite video games, if you didn't notice my background poster! Dancing to me is very important, because it always my brain to get out of working mode and drift into a more creative side. Apart from expressing mental creativity, it also keeps me in great shape. Hip hop dance is my most practiced type of dance. I also enjoy salsa dance.

My Projects

Online PC store

Shop PC parts is a simple online store where one may computer parts. This project is mostly a front end design and doesn't have any backend functionality just yet to allow for actual purchasing. The user can may create an account, search the inventory for items to add to the cart, and can view the cart items page.

Ticket Support System

This site allows the user to create an account and submit tickets, which are essentially requests to have something fixed or resolved. A ticketing system is used often in the work field to let departments, such as IT, aware of technical issues. Some examples are problems with user login authentication, or a hardware issues.

Weather Park Search

Walk in the Park is an application which allows the user to search up nearby parks with weather results combined. The idea of this project was to help the user decide if visiting a certain park could be a pleasant experience based on what the weather is like for a certain day.
